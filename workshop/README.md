# AI-DLC Workshop Materials

This directory contains comprehensive workshop materials for learning and applying the AI-DLC (AI-Driven Development Life Cycle) workflow.

## Workshop Overview

The AI-DLC workshops provide hands-on experience with the complete AI-DLC workflow, from discovery through testing. Choose the workshop format that best fits your needs:

- **2-Day Workshop**: Condensed introduction for teams new to AI-DLC
- **5-Day Workshop**: Comprehensive, in-depth coverage for production adoption

## Workshop Structure

### Core Materials

1. **Installation Guide** (`00-installation-guide.md`)
   - Step-by-step setup instructions
   - Platform-specific configuration
   - Verification steps

2. **Workshop Guides**
   - **2-Day Workshop** (`01-workshop-2day.md`)
   - **5-Day Workshop** (`02-workshop-5day.md`)

3. **Phase Instructions** (`phase-instructions/`)
   - Detailed instructions for each phase
   - Example inputs for phase transitions
   - Tips and best practices

4. **Report Templates** (`report-templates/`)
   - Discovery Report
   - Business Vision Report
   - Scope Report
   - Design Compatibility Report

## Workshop Phases

The workshops cover six main phases:

1. **Discovery**: Establish environment, coding standards, Agile process
2. **Vision**: Business vision and MVP scope
3. **Units and User Stories**: Problem breakdown and story creation
4. **Design**: Complete design review
5. **Code Development**: AI builds the code
6. **Testing**: Test plan, harness, and execution

## Phase Transitions

When transitioning between phases, use the example phase boundary inputs provided in each phase instruction file. These inputs clearly communicate phase completion and readiness to proceed.

**Example:**
```
Discovery phase complete. Environment established, coding standards defined, Agile process documented. Ready to proceed to Vision phase.
```

## Common Questions

**Q: Can I customize the workshop?**
A: Yes, adapt the workshops to your team's needs and timeline.

**Q: What if we don't finish a phase?**
A: Document progress and continue in the next session. Use phase boundary inputs to resume.

**Q: Can we use our own project?**
A: Yes, the workshops work with any project. Choose a project appropriate for the workshop duration.

**Q: What if we have questions?**
A: Refer to the phase instructions, report templates, and main AI-DLC documentation.
